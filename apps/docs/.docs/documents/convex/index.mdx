# Convex

Convexは、アプリ開発者のためのオープンソースのリアクティブデータベースです。クエリはデータベース内で直接実行されるTypeScriptコードであり、Reactコンポーネントが状態変化に反応するのと同様に、Convexクエリはデータベースの変更に反応します。

## Convexとは

Convexは、データベース、サーバー関数を記述する場所、そしてクライアントライブラリを提供します。動的でリアルタイム更新するアプリケーションの構築とスケーリングを容易にします。

Convexの主な特徴：

- **すべてがコード** - データベーススキーマからクエリ、認証からAPIまで、バックエンドのあらゆる部分を純粋なTypeScriptで表現できます
- **常に同期** - フロントエンドコード、バックエンドコード、データベース状態の変更がリアルタイムで反映されることを保証します
- **お気に入りのフレームワークとの連携** - React、Next.js、React Native、Vue、Svelte、Pythonなど、多くのフレームワークと連携可能です

## 主要機能

### リアクティブデータベース

データベースの変更に自動的に反応するクエリを作成できます。状態管理、キャッシュ無効化ポリシー、WebSocketの設定が不要です。

### TypeScriptネイティブ

バックエンドコードをTypeScriptで記述し、型チェックと自動補完の恩恵を受けられます。AIによる高精度なコード生成も可能です。

### リアルタイム同期

クライアントライブラリがアプリケーションの状態をリアルタイムで同期し、常に最新の状態を維持します。

### 認証と権限

組み込みの認証システムと柔軟な権限モデルにより、セキュアなアプリケーションを構築できます。

### スケジュールされたタスク

定期的なタスクやバックグラウンド処理を簡単にスケジュールできます。

### ファイルストレージ

ファイルのアップロード、保存、取得を簡単に行えます。

### 検索機能

データベース内のデータに対する高度な検索機能を提供します。

## 始め方

Convexを使い始めるには、公式ドキュメントに従って環境をセットアップし、最初のアプリケーションを構築してください。

Convexは、クラウドプラットフォームを通じて利用するのが最も簡単な方法です。無料枠が寛大に用意されており、インフラストラクチャを気にすることなくアプリケーションの構築に集中できます。多くの小規模アプリケーションやサイドプロジェクトは、完全に無料で運用できます。

## セルフホスティング

Convexはセルフホスティングも可能です。Docker（推奨）または事前にビルドされたバイナリを使用してConvexをセルフホストできます。セルフホスティングガイドに詳細な手順が記載されています。
