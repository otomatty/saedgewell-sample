# Linear へようこそ

Linear は、チームが迅速に行動し、組織を維持し、効率的に作業できるように設計されています。プロジェクトの管理、課題の追跡、開発プロセスの最適化など、Linear はチームのニーズに適応するクリーンで直感的なプラットフォームを提供します。このガイドでは、Linear の主要な機能をご紹介し、迅速にセットアップできるようサポートします。重要なことは、影響力のある仕事を提供することです。さあ、始めましょう！

# Linear の概要

Linear の環境の概要を把握するには、Linear の概要ビデオをご覧ください。

# Linear デモ

Linear の一部の機能とワークフローを試すには、[デモインスタンス](https://linear.app/demo)にアクセスしてください。行った更新は Web ブラウザに対してローカルであり、ページを更新するとクリアされます。

試せること:

- プロジェクトリストと P2P App プロジェクトを開き、ドキュメントの作成、箇条書きリストを課題に変換、マイルストーンの作成などのプロジェクト機能を試してください。
- ショートカット C を使用して、任意のビューから課題を作成します。

デモでは設定を探索できず、SLA をサポート/表示しないという制限がありますが、概要を把握するのに役立ちます。

[ライブオンボーディング セッションに参加](https://lu.ma/welcome-to-linear?utm_source=docs)して、Linear の基本を学び、主要なワークフローのデモをご覧ください。

# Linear を組織に適用する

概要を把握し、Linear の一部の機能の探索を開始したので、Linear が組織でどのように機能するかを確認しましょう！

これを行う最良の方法は、(できれば) 職場のメールアドレスを使用してサインアップして、ワークスペースを作成することです。

ワークスペースを作成したら、オンボーディングするユーザー向けに Linear の一般的な概要を提供する [Linear を最大限に活用するためのガイド](https://linear.app/docs/making-the-most-of-linear)をご覧ください。

# チームを Linear にオンボーディングする

チームを Linear にオンボーディングするには、ワークスペース管理者の多くのガイダンスが必要です。

その負担を軽減するために、新しい Linear ワークスペースに参加するチームメンバー向けのヒント [Linear でチームに参加するためのヒント](https://linear.app/docs/onboarding-your-team-on-linear) を提供しています。

他のツールから切り替えるためのリソースを [スイッチ取扱説明書ページ](https://linear.app/docs/switch)にまとめました。
