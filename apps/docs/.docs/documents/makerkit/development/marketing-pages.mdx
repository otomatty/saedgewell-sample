# Next.js Supabase Turboスターターキットのマーケティングページ

Makerkitには、SaaSアプリケーションの開始に役立つ事前定義されたマーケティングページが付属しています。これらのページはNext.jsとTailwind CSSで構築されており、`apps/web/app/(marketing)`ディレクトリに配置されています。

Makerkitには以下のマーケティングページが含まれています：

- ホームページ
- お問い合わせフォーム
- 料金ページ
- よくある質問（FAQ）
- お問い合わせページ（お問い合わせフォーム付き）

## 新しいマーケティングページの追加

Makerkitアプリケーションに新しいマーケティングページを追加するには、以下の手順に従う必要があります。

`apps/web/app/(marketing)`ディレクトリに、ページに使用したいパスのフォルダを作成します。例えば、`/about`に新しいページを作成するには、`about`という名前のフォルダを作成します。次に、そのフォルダ内にページファイルを作成します。例えば、aboutページを作成するには、aboutフォルダ内に`page.tsx`ファイルを作成します。

```tsx
// apps/web/app/(marketing)/about/page.tsx
export default function AboutPage() {
  return <div></div>
}
```

このページは`apps/web/app/(marketing)/layout.tsx`のレイアウトを継承します。このファイルを編集してレイアウトをカスタマイズできますが、それがすべてのマーケティングページに影響することを覚えておいてください。

## お問い合わせフォーム

お問い合わせフォームを機能させるには、以下の環境変数を追加する必要があります：

```
CONTACT_EMAIL=
```

この変数には、お問い合わせフォームの送信を受け取りたいメールアドレスを設定する必要があります。送信者はメール設定で構成したものと同じになります。

## マーケティングページのカスタマイズ

マーケティングページは通常のNext.jsページとして機能するため、必要に応じて自由にカスタマイズできます。以下は一般的なカスタマイズの例です：

1. **ブランディングの更新**: ロゴ、色、フォントなどを自社のブランドに合わせて変更します
2. **コンテンツの追加**: 製品の特徴、顧客の声、使用例などを追加します
3. **SEO最適化**: メタタグ、OGタグ、構造化データなどを追加して検索エンジン最適化を行います

## マーケティングページのコンポーネント

Makerkitには、マーケティングページで使用できる様々なコンポーネントが含まれています。これらのコンポーネントは`@kit/ui`パッケージから利用できます。

例えば、ヒーローセクションを作成するには：

```tsx
import { Container } from '@kit/ui/container';
import { Heading } from '@kit/ui/heading';

export function HeroSection() {
  return (
    <Container>
      <div className="py-24 text-center">
        <Heading level={1}>あなたの製品名</Heading>
        <p className="mt-4 text-lg text-gray-600">
          あなたの製品の簡単な説明をここに書きます
        </p>
      </div>
    </Container>
  );
}
```

これらのコンポーネントを組み合わせて、魅力的なマーケティングページを作成できます。
