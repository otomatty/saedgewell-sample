# Next.js Supabase SaaS Kit Turboリポジトリの紹介

Next.js Supabase SaaS Kit Turboリポジトリへようこそ。
このリポジトリは、[[Supabase]]でSaaSアプリケーションを構築するためのNext.jsスターターキットです。

Next.js Supabase SaaS Kitの最新バージョンへようこそ！このスターターキットは、Next.jsとSupabaseの強力な組み合わせを使用して、堅牢なSaaSアプリケーションを構築するのに役立つように設計されています。

初心者でも経験豊富な開発者でも、このドキュメントはSaaSアプリケーションのセットアップ、設定、デプロイについてガイドします。

## 新機能 🎉

以前のバージョンと比較して、このキットには以下が含まれています：

- **より多くの機能**: 現代のSaaS要件を満たすための拡張機能
- **整理されたコード**: より整理され、保守しやすいコードベース
- **最新の依存関係**: 最新のライブラリとツール

## 機能

### 認証
- **サインアップ、サインイン、サインアウト**: 簡単で安全なユーザー認証
- **パスワードを忘れた場合**: シンプルなパスワード回復プロセス

### 多要素認証（MFA）
- **MFAの有効化**: アカウントにセキュリティの追加レイヤーを追加

### アカウント管理
- **個人アカウント**: ユーザーは組織アカウントに加えて、独自の個人アカウントを持っています
- **組織アカウント**: ユーザーは組織を作成して参加できます
- **組織の役割**: 組織内のユーザーに異なる役割を割り当てます
- **組織の招待**: 他のユーザーを組織に招待します

### 課金
- **個人アカウントの課金**: 個人アカウントの課金情報を管理
- **組織の課金**: 組織アカウントの課金を処理
- **サブスクリプション**: フラット、階層型、座席ごとのモデルなど、さまざまなサブスクリプションプランをサポート
- **複数の課金プロバイダー**: Stripe、Lemon Squeezy、（近日公開）Paddleのすぐに使えるサポート

### コンテンツ管理システム（CMS）
- **複数のCMSサポート**: KeystaticとWordPressとの統合

### モニタリング
- **モニタリングツール**: SentryとBaselimeの組み込みサポート

## モジュール式、拡張可能、そして最新のSaaSスターターキット

### モジュール構造
このキットの最大の変更点はモジュール構造であり、以下を可能にします：

- 🧩 **簡単な機能統合**: 最小限の変更で新機能をキットにプラグイン
- 🛠️ **簡素化された保守**: コードベースをクリーンで保守しやすく保つ
- 🎯 **コア機能の分離**: コア機能とカスタム機能を区別
- 🌟 **追加モジュール**: 課金、CMS、モニタリング、ロガー、メーラーなどのモジュールを簡単に追加
