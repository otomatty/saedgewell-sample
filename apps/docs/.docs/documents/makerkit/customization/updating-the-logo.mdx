# Makerkitアプリケーションでのデフォルトロゴの更新

もちろん、アプリケーションに独自のロゴを使用したいでしょう。そのためには、`apps/web/components/app-logo.tsx`にあるコンポーネントを自分のロゴで更新してください。

SVGコンポーネントを使用するか、画像をドロップするか、実際には何でも構いません。このコンポーネント内でレンダリングされる限り、どのように実装しても問題ありません。

ロゴ画像は、認証ページ、サイトヘッダー、サイトフッター、アプリサイドバー（チームアカウントが無効の場合）など、アプリケーション全体で使用されます。
