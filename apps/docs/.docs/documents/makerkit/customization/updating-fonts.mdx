# SaaSのデフォルトフォントを更新する

デフォルトでは、Makerkitは2つのフォントを使用しています：

- **font-sans**: Appleデバイスではアップルのデフォルトフォント、その他ではInterを使用
- **font-heading**: Google FontsのUrbanistを使用

フォントは`apps/web/lib/fonts.ts`で定義されています：

```tsx
import { Urbanist as HeadingFont, Inter as SansFont } from 'next/font/google';

/**
 * @sans
 * @description Define here the sans font.
 * By default, it uses the Inter font from Google Fonts.
 */
const sans = SansFont({
  subsets: ['latin'],
  variable: '--font-sans',
  fallback: ['system-ui', 'Helvetica Neue', 'Helvetica', 'Arial'],
  preload: true,
  weight: ['300', '400', '500', '600', '700'],
});

/**
 * @heading
 * @description Define here the heading font.
 * By default, it uses the Urbanist font from Google Fonts.
 */
const heading = HeadingFont({
  subsets: ['latin'],
  variable: '--font-heading',
  fallback: ['system-ui', 'Helvetica Neue', 'Helvetica', 'Arial'],
  preload: true,
  weight: ['500', '700'],
});

// we export these fonts into the root layout
export { sans, heading };
```

異なるフォントを表示するには、フォントがGoogle Fontsにある場合は`next/font/google`からのインポートを置き換えてください。

## AppleデバイスでのAppleシステムフォントをデフォルトから変更する

Appleのシステムフォントの代わりに別のフォントを設定するには、Tailwind変数を更新します。

`apps/web/styles/shadcn-ui.css`を開き、`-apple-system`を使用したいフォントに置き換えてください。
