## キーボードショートカット

| 操作 | ショートカット (Windows/Linux) | ショートカット (macOS) |
|------|------------------------------|----------------------|
| デフォルトタスク実行 | `Ctrl+Shift+B` | `Cmd+Shift+B` |
| タスク選択メニュー | `Ctrl+Shift+P` → "Tasks: Run Task" | `Cmd+Shift+P` → "Tasks: Run Task" |

- デフォルトタスクは「🚀 開発環境: すべて起動」に設定されています。
- それ以外のタスクに関しては `Ctrl+Shift+P` を使用してください。

## よく使うタスク

### 基本的な開発サイクル

```
🚀 開発環境: すべて起動  →  （開発作業）  →  🛑 開発環境: すべて停止
```

### アプリ個別起動

```
🖥️ 開発環境: Webアプリのみ起動
🖥️ 開発環境: Docsアプリのみ起動
🖥️ 開発環境: Adminアプリのみ起動
```

### ログ監視

```
📋 ログ: Webアプリ
📋 ログ: 全サービス
```

## タスクカテゴリ別早見表

### 起動関連 🚀

- **🚀 開発環境: すべて起動** - 環境をフルセットアップして起動
- **🖥️ 開発環境: アプリのみ起動** - 全アプリを起動
- **💻 開発環境: アーキテクチャ最適化起動** - CPUタイプに最適化して起動
- **📊 Supabase: 起動** - Supabaseのみ起動

### 停止関連 🛑

- **🛑 開発環境: すべて停止** - すべてのサービスを停止
- **🧹 開発環境: すべて停止してボリューム削除** - データをリセットして停止

### セットアップ 🔧

- **🔄 初期セットアップ実行** - 新環境の初期セットアップ（ホスト名 → HTTPS証明書 → シークレット）
- **🔧 セットアップ: ホスト名** - ローカルドメイン設定
- **🔒 セットアップ: HTTPS証明書** - SSL証明書生成

### トラブルシューティング 🔍

- **📋 確認: Supabase状態** - Supabase実行状態確認
- **🔍 診断: JWT認証** - JWT認証問題診断
- **🔧 修正: PostgREST** - PostgREST問題修正

## 一般的なユースケース

### 新しいマシンでの初回セットアップ

```
🔄 初期セットアップ実行  →  🚀 開発環境: すべて起動
```

### 環境リセット（データベースクリーン）

```
🧹 開発環境: すべて停止してボリューム削除  →  🚀 開発環境: すべて起動
```

### 環境の再起動

```
🔄 再起動
```

### JWT認証問題の解決

```
🔍 診断: JWT認証  →  🔐 セットアップ: JWTシークレット設定
```
