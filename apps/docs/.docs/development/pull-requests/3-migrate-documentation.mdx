# ドキュメント構造の移行と改善

## 概要

このプルリクエストでは、ドキュメントをルートディレクトリの `.docs` に移行し、関連するドキュメント構造と機能を改善しました。また、コントリビューター情報や使用ライブラリ、デザインインスピレーションに関する新しいドキュメントを追加し、プロジェクトの透明性と情報の可視性を向上させました。

## 主な変更点

- ドキュメントを `apps/docs/contents` から `.docs` ディレクトリに移行
- コントリビューター、使用ライブラリ、デザインインスピレーションに関する新しいドキュメントを追加
- 各ドキュメントの構造を整理し、情報の可視性を向上
- 検索機能の実装を改善し、エラーハンドリングを強化
- ヘッダーとフッターのレイアウトを更新し、ナビゲーションを強化
- 見出しのスタイルを調整し、可読性を向上

## 技術的詳細

ドキュメントをルートディレクトリに移行することで、プロジェクト全体での参照がしやすくなり、ドキュメント管理がより効率的になりました。また、検索機能の改善により、ユーザーがドキュメント内の情報をより簡単に見つけられるようになりました。
