# 謝辞

このプロジェクトは多くの個人、オープンソースプロジェクト、ライブラリ、そしてデザインインスピレーションの助けを借りて実現しています。このページでは、それらすべてに対する感謝の意を表します。

## コントリビューター

このプロジェクトは、以下の方々の貢献によって成り立っています。

### メインテナー

- **[otomatty](https://github.com/otomatty)**
  - プロジェクトの立ち上げ
  - コアアーキテクチャの設計
  - メインテナンス

### コントリビューター

プロジェクトの改善に貢献してくださった方々：

- 機能追加、バグ修正、ドキュメント作成、翻訳など、様々な形でプロジェクトに貢献してくださった方々のリストは[contributors.md](./contributors)でご確認いただけます。

## 使用ライブラリ

このプロジェクトは、以下の主要なライブラリとフレームワークを活用しています：

### フロントエンド

- **Next.js** (v15.x) - アプリケーションフレームワーク
- **Tailwind CSS** (v4.x) - UIスタイリング
- **Shadcn UI** - UIコンポーネント

### バックエンド

- **Supabase** - バックエンドサービス（認証、データベース、ストレージ）

### 開発ツール

- **Turborepo** - モノレポ管理
- **Biome** - コード品質管理

その他のライブラリと詳細な活用方法については、[libraries.md](./libraries)をご参照ください。

## デザインインスピレーション

このプロジェクトのデザインや機能は、以下のプロジェクトから大きなインスピレーションを受けています：

- **Create T3 App** - 型安全性を重視したアプローチ
- **Taxonomy** - UIコンポーネントの設計とアクセシビリティ
- **Vercel Templates** - モダンなデザインとユーザー体験
- **Next.js Examples** - 機能実装のベストプラクティス
- **Supabase Templates** - バックエンド設計とセキュリティ実装
- **Turborepo Examples** - モノレポ構成とビルド最適化

各プロジェクトからの具体的なインスピレーションと採用したパターンについては、[inspiration.md](./inspiration)で詳しく紹介しています。

## コントリビューション方法

このプロジェクトへの貢献に興味をお持ちの方は、以下の手順で参加いただけます：

1. [Issues](https://github.com/otomatty/super-next-app/issues)をチェック
2. 取り組みたいIssueにコメント
3. フォークしてプルリクエストを作成
4. レビュープロセスに参加

詳細な貢献ガイドラインについては、[CONTRIBUTING.md](../../CONTRIBUTING.md)をご覧ください。

このプロジェクトは[Contributor Covenant](https://www.contributor-covenant.org/)の行動規範に従っています。
詳細は[CODE_OF_CONDUCT.md](../../CODE_OF_CONDUCT.md)をご覧ください。
