# 印刷会社向けホームページ要件定義書

## 1. 概要

### 1.1 目的
本ホームページは、ニイヌマ企画印刷の企業情報、サービス内容、実績を効果的に発信し、新規顧客獲得と既存顧客への情報提供を行うためのウェブサイトです。また、業務管理システムとWebアプリケーション機能への入口としての役割も担います。

### 1.2 対象ユーザー
- 印刷サービスを検討している潜在顧客
- 既存の顧客
- 企業情報を調査する取引先・パートナー
- 採用情報を探す求職者

### 1.3 開発方針
- モダンなデザインと直感的なUIを採用
- レスポンシブデザインによる多デバイス対応
- SEO対策を徹底し、集客力の高いサイト構築
- 業務システム・Webアプリケーションとのシームレスな連携

## 2. 機能要件

### 2.1 コンテンツページ構成

#### 2.1.1 トップページ
- メインビジュアル（スライダー形式）
- サービス概要（サムネイル付きカード形式）
- 新着情報・お知らせ（最新5件）
- 制作実績ハイライト（最新3件）
- お客様の声（ランダム表示3件）
- 問い合わせへの導線
- 業務システム・Webアプリへのログインリンク

#### 2.1.2 会社情報ページ
- 企業理念・ビジョン
- 会社概要（設立年、資本金、従業員数など）
- 沿革
- 代表メッセージ
- アクセス情報（地図埋め込み）
- 事業所一覧

#### 2.1.3 サービス紹介ページ
- サービス一覧（カテゴリ別）
  - オフセット印刷
  - デジタル印刷
  - 大判印刷
  - 加工サービス
  - デザインサービス
- 各サービス詳細ページ
  - サービス説明
  - 特長・メリット
  - 対応サイズ・仕様
  - 料金目安
  - 制作フロー
  - 関連サービスリンク

#### 2.1.4 制作実績ページ
- 実績一覧（カテゴリ・業種でフィルタリング可能）
- 実績詳細ページ
  - 制作物画像（複数枚）
  - 案件概要
  - 採用した印刷・加工技術
  - お客様の評価
  - 関連実績リンク

#### 2.1.5 お客様の声ページ
- 顧客レビュー一覧（業種別・サービス別にフィルタリング可能）
- 評価スコア（星評価）
- 詳細コメント
- お客様プロフィール（業種、地域など）

#### 2.1.6 料金シミュレーターページ
- 印刷種類選択
- サイズ・部数・紙質選択
- オプション（加工）選択
- 納期設定
- 料金自動計算
- 見積書PDF出力機能
- 問い合わせへの導線

#### 2.1.7 お問い合わせページ
- 問い合わせフォーム
  - 個人情報（氏名、会社名、連絡先）
  - 問い合わせ内容（選択式）
  - 詳細メッセージ（自由記述）
  - 添付ファイル機能
- よくある質問（FAQ）セクション
- 電話・メール等の直接連絡先情報

### 2.2 共通機能

#### 2.2.1 ヘッダー
- ロゴ
- グローバルナビゲーション
- お問い合わせボタン
- ログインボタン（業務システム・Webアプリ用）
- 検索機能

#### 2.2.2 フッター
- サイトマップ
- 会社情報サマリー
- お問い合わせ情報
- SNSリンク
- プライバシーポリシー・特定商取引法に基づく表記へのリンク
- コピーライト

#### 2.2.3 その他共通機能
- パンくずリスト
- ページトップへ戻るボタン
- クッキー通知・同意バナー
- モバイルナビゲーション（ハンバーガーメニュー）

## 3. 非機能要件

### 3.1 デザイン要件
- コーポレートカラーを基調としたデザイン
- 印刷品質の高さを感じさせる洗練されたビジュアル
- 写真・画像は高解像度で印刷物の質感が伝わるもの
- タイポグラフィを重視した読みやすいレイアウト
- アニメーション効果は控えめに使用（ページ読み込み速度優先）

### 3.2 技術要件
- HTML5/CSS3/JavaScript（最新標準準拠）
- フロントエンドフレームワーク：Next.js（React）
- レスポンシブ対応：モバイル、タブレット、デスクトップ
- CMS機能：コンテンツ管理システム実装
- ブラウザ対応：Chrome、Firefox、Safari、Edge最新版
- ページ読み込み速度：モバイルで3秒以内
- AMP対応（モバイルページ高速化）

### 3.3 SEO要件
- 全ページにメタタグ最適化
- OGPタグ実装（SNS共有対応）
- サイトマップXML自動生成
- Google Search Consoleとの連携
- 構造化データ（Schema.org）の実装
- キーワード最適化（印刷関連キーワードリサーチに基づく）
- モバイルフレンドリー対応

### 3.4 セキュリティ要件
- SSL/TLS証明書導入（HTTPS対応）
- XSS、CSRF対策
- Webアプリケーションファイアウォール（WAF）導入
- コンタクトフォームのスパム対策（reCAPTCHA等）
- 個人情報入力時の安全性確保
- クッキーポリシー対応

### 3.5 パフォーマンス要件
- 画像最適化（WebP形式サポート）
- キャッシュ戦略の実装
- CDN（コンテンツデリバリーネットワーク）の活用
- ページ表示速度のGoogle PageSpeed Insightsスコア90以上
